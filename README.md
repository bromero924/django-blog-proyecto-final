# Proyecto Django - Blog Sabor Santanderiano

## Descripción

Aplicación web desarrollada en Django que permite:

- Registro de usuarios
- Login y logout
- Perfil de usuario
- Edición de perfil
- Cambio de contraseña
- Sistema de mensajería entre usuarios
- CRUD de páginas (blog)

## Tecnologías utilizadas

- Python
- Django
- SQLite
- HTML
- CSS

## Funcionalidades

Home  
About  
Pages (CRUD)  
Signup  
Login  
Logout  
Perfil de usuario  
Editar perfil  
Cambio de contraseña  
Sistema de mensajes  

## Instalación

Clonar repositorio:
```
git clone URL_DEL_REPOSITORIO
```

Instalar dependencias:

```
pip install -r requirements.txt
```

Aplicar migraciones:

```
python manage.py migrate
```

Ejecutar servidor:

```
python manage.py runserver
```
